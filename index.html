What is Entitybase?

Entitybase is a platform for creating APIs
(not just documenting or mocking API responses like others out there).
With Entitybase, you give your API a name, give it some resources
(e.g. users, posts, tags and comments) and, just like magic, you're good to go.
What used to take a team of developers months can now be done in minutes.
