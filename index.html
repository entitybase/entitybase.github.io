What is Entitybase?

How Entitybase works

1. Create a new API

Creating an API is as easy as choosing a name, a web address, and clicking Create.

2. Create a resource

A resource is a kind of data - e.g. customers, products, articles or photos.

3. Create an attribute

An attribute is something a resource has - e.g. a customer has a name and a product has a price.

4. Review your docs

Head over to your new API documentation for easy-to-follow instructions on how to create, read, update and delete data in your API.

5. Start using your API

That's it. In less than a minute, your new API is ready to use for anything you like.
