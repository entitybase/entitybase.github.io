How Entitybase works

1. Create a new API

Give your API a name and a URL. Every Entitybase API get's it's own web address.

2. Create some resources

Every good API needs resources. A resource is a kind of data - e.g. users, articles, products or photos.

3. Create some attributes

Attribute are the things that a resource has - e.g. a user has a name, an article has a title and a product has a price.

4. Check out your docs

Head over to your API documentation for easy-to-follow instructions on how to create, read, update and delete data in your API.

5. Start using your API

That's it. Your API is ready to use for anything you like.
